Video Tutorials

General

Below are video tutorials that you can use to become familiar with SimPatient. The first 6 are for the Administrator Workflow. While they are in a particular order, a lot of you can can do from an adminstrator account is non-linear. The 7th video is for the Station Workflow, which is very linear, and justified being contained in a single video.

2 more videos were added. Video 8 is for the Medication Administration Record Archive Viewer accessible via an administrator account. Video 9 is a miscellaneous tutorial covering features that were added to the application to round it out before completion. This covers a User Acccount password change feature, a Generate MAR Sheet button added to the Simulation Editor Control, and the delete functionality that has been enabled application-wide.
